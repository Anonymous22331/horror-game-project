# horror-game-project
Тестовый проект в жанре хоррор в стиле Fears of Fathom. Проект закрыт.
Небольшой сюжет, рассчитанный на 5–10 минут.
В рамках проекта проводилось тестирование Unity 6, а также работы со светом, звуком и общей атмосферой. Реализованы простые игровые механики и базовые анимации.
Проект выполнен примерно за 8 часов, в формате быстрого прототипа.
